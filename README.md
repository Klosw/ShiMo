# ShiMo
用于石墨文档下载的java程序 多cookie(确保权限一致) 可能会提高下载速度 

能下载石墨绝大部分的文档以及文件 

使用方式 电脑上安装java11 

查看帮助 
```
java -jar ShiMo.jar -h 
```
使用
```
java -jar ShiMo.jar -c s%*******1 s%*******2 s%*******3 -f 文件夹(卡通过下图获取) 
```
