# ShiMo
用于石墨文档下载的java程序

使用方式 电脑上安装java11 

查看帮助 
```
java -jar ShiMo.jar -h 
```
使用
```
java -jar ShiMo.jar -c s%*******1 s%*******2 s%*******3 -f 文件夹ID 
```
